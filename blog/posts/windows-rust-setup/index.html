Setting up rust on Windows

2022-01-13
2 minute read
rust
rust , windows , openssl

Note: there’s no actual solution here; just a list of attempts that fix one bit but then lead straight into the next breaking package. the openssl build fix worked fine.

Setting up rust on Windows is surprisingly easy these days; right up till it isn’t. So first; prerequisites:

  • Microsoft have a nice setup doc here https://learn.microsoft.com/en-us/windows/dev-environment/rust/setup
  • Where I use paths that contain Visual Studio you should substitute that for build-tools paths as needed.
  • Microsofts page says Rust works very well on Windows; so there's no need for you to go the WSL route; try not to choke on your brew

Unfortunately quite a lot of products in the rust world require recompiling openssl; which is of course where the headaches begin.

We’ll use vector as an example; if you don’t know what vector is, its a multi-source in multi-source out metrics and log data agent that you can run to pump metrics and logs, transform them, and send to multiple destinations. Neat. It’s written in rust… good for performance*, bit of a pain to get quickly up and running to change bits when you’re on Windows.

Requires CMake; cool, I’m a .net dev so my machine already has VS installed, the C++ desktop tools and the linux c++ dev tools which includes cmake. … Can’t access cmake Ah; but Microsoft have installed it at a path so damn deep it wont run from path alone. Greeeeeat.

So we create a folder like, say, c:\tools\ and install cmake there from https://cmake.org/download/ quick and easy. Now we can add it to the path setx cmk c:\CMake\bin /m (admin prompt required, or use the environment ui under system) Add to your path setx /M PATH "%PATH%;%cmk%" Create a new console and type cmake --version it should be in path… yey.

Deep breath, next we need to appease other gods; one of which is perl. Luckily Microsoft has saved the day here and we have vcpkg; so lets grab that.

 1cd c:\tools
 2git clone https://github.com/microsoft/vcpkg
 3cd vcpkg
 4bootstrap-vcpkg.bat
 5vcpkg install openssl-windows:x64-windows openssl:x64-windows-static
 6REM https://docs.rs/vcpkg/latest/vcpkg/
 7setx VCPKGRS_DYNAMIC 1 /m
 8setx VCPKG_ROOT c:\tools\vcpkg
 9setx OPENSSL_NO_VENDOR 1
10setx RUSTFLAGS -Ctarget-feature=+crt-static /m
11REM you dont technically need both the env and integrate call
12vcpkg integrate install
13mkdir c:\tools\opensslcert\
14wget https://curl.se/ca/cacert.pem -o c:\tools\opensslcert\cacert.pem
15setx SSL_CERT c:\tools\opensslcert\cacert.pem /m
