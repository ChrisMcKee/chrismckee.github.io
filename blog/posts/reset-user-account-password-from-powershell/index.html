Resetting Windows Server User Passwords

2023-03-07
0 minute read
windows-server
windows-server , server-core , powershell

Quick and simple

This is also useful for AWS when you have SSM setup but the servers created without a key; as it allows you to set the password via ssm then RDP normally.

  • Remote to the machine / open powershell as admin
  • Check current user Get-LocalUser
  • Create new password you’ll use, as a secure-string $Password = Read-Host "Enter the new password" -AsSecureString
  • Set the password of the user. (in this case admin)
1$UserAccount = Get-LocalUser -Name "administrator"
2$UserAccount | Set-LocalUser -Password $Password

Done
